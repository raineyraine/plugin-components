plugin-components is made to make creating Roblox Studio plugins with React in
Roblox-TS easier. Instead of spending time making components, focus on making
the actual plugin.

- Useful React Hooks for plugins
- Pre-made, Roblox Studio-like components
- Components for Studio plugin behavior (toolbars, buttons, windows)
- Other plugin utilities

### Installation

> [!IMPORTANT]  
> [rbxts-react](https://github.com/littensy/rbxts-react/tree/main) should be
> installed already.

Install plugin-components from
[npm](https://npmjs.com/package/@rbxts/plugin-components):

```sh
npm install @rbxts/plugin-components
pnpm add @rbxts/plugin-components
yarn add @rbxts/plugin-components
```
