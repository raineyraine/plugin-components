# Installation

[@rbxts/plugin-components](npmjs.com/package/@rbxts/plugin-components "@rbxts/plugin-components on npm")
can be installed with any package manager from npm.

::: code-group

```sh [npm]
npm i @rbxts/plugin-components
```

```sh [pnpm]
pnpm add @rbxts/plugin-components
```

```sh [yarn]
yarn add @rbxts/plugin-components
```

:::

::: danger

plugin-components is not published on npm yet. Installation will fail.

:::
